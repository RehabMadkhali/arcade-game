Frogger Arcade Game
===============================
# Summary

Frogger Arcade is a variation of the classic Frogger arcade game.

## Game Goal:
get the player to other side (to the Water) without colliding with enemies ( Bugs!)

## Game Rules:
- player should avoid the bugs.
- player should reach the water.
